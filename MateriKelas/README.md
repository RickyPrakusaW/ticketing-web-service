# Inisialisasi

Untuk menginisialisasi program kita (menghasilkan package.json)
`npm init -y`

# Extension VSCODE

- prettier
- ES7 punya dsznajder
- node-snippets punya chris noring

# Install nodemon

`npm i -g nodemon`

# Cara menjalakan

- node index.js
- npx nodemon index

# Cara akses di echo Api / postman

- Pastikan HTTP Verb betul (GET | POST | PUT | PATCH | DELETE)
- url diawali dengan domain:port
  ` localhost:3000`
- Ikutin dengan route, baca kodingan
